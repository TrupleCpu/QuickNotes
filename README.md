# QuickNotes

## What is QuickNotes?
 QuickNotes is an online notepad that uses the slatejs library to build a rich text much better than the traditional <textarea>.

### Front-End language used
  - Typescript
#### Libariries used
   - Slatejs
   - axios
   - slate-history
   - aos
   - framer-motio
   - react-helmet
   - react-icons
   - react-to-print
   - react-spinners
   - react-router-dom
 
## Back-End language used
  - Javascript
